파이어베이스로 서버를 만드려고 했으나 능력과 시간 부족으로 서버 구축에 실패하여 로컬에서만 작동합니다..
실행환경이 Android -> Console 로 변경되었습니다.


시스템 소개: 친구를 맺은 사이에서 서로의 생존 여부를 확인할 수 있는 프로그램 입니다.


* 사용 방법(시나리오) *

- 텍스트의 입출력으로 작동합니다.
- 대괄호 안에 적힌 텍스트를 입력해주세요. 


1. [2]를 입력하여 회원가입을 진행해주세요. 

2. [1]을 입력하여 로그인 후 [친구코드] 를 확인하고 복사해주세요.

3. [로그아웃] 해주세요.

4. 테스트를 위한 회원정보를 저장해두었습니다. [1] 로그인을 시도해주세요
     > id : test 
        pw: test

5. 로그인에 성공하였다면 도착한 친구요청이 출력됩니다.
   [수락] 또는 [거절]을 입력하여 친구요청을 처리해주세요.

6. [새로고침] 을 입력하면 화면이 새로고침 됩니다.

   [생존신고] 를 입력하면 마지막 생존신고일이 현재시각으로 갱신됩니다.

   [프로필] 을 입력하면 이름을 변경할 수 있습니다.

   [친구의 이름] ex)홍길동 을 입력하면 친구에게 신고 [요청]을 하거나 친구를 [삭제]할 수 있습니다.

   [친구추가] 를 눌러 (2)에서 복사해둔 친구코드를 입력해주세요. 친구요청이 전송됩니다.

   [로그아웃] 하여 이전에 가입해둔 id로 다시 로그인 하면 테스터에게서 온 친구요청이 있습니다.
